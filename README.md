# Maggie The MUA 💄

A comprehensive web application for Maggie's professional makeup artistry services and premium hair weave store. Clients can book makeup appointments and purchase quality hair extensions seamlessly.

## ✨ Features

### 💅 Makeup Services
- **Professional Booking System**: Schedule appointments for various makeup services
- **Service Customization**: Add-ons like lashes, facials, and eyebrow shaping
- **Quantity Selection**: Book for individuals or groups
- **Package Deals**: Special wedding and event packages with mobile services
- **Flexible Scheduling**: Choose dates and times that work for you

### 👑 Hair Weave Store
- **Premium Hair Collection**: Shop quality weaves and extensions
- **Product Catalog**: Browse different hair types, lengths, and styles
- **Secure Checkout**: Safe payment processing for hair purchases
- **Inventory Management**: Real-time stock availability

### 🔒 Payment & Security
- **South African Payment Gateways**: PayFast and Yoco integration
- **Deposit System**: Secure booking fees to reserve appointments
- **Payment Security**: Encrypted transactions and data protection

### ⚙️ Admin Features
- **Appointment Management**: View, manage, and update bookings
- **Service Catalog**: Add and edit services and prices
- **Inventory Control**: Manage hair weave stock levels
- **Customer Management**: Track client history and preferences

## 🛠️ Tech Stack

### Frontend
- **React.js** - Modern UI framework
- **TailwindCSS** - Utility-first CSS framework
- **Axios** - HTTP client for API calls
- **React Router** - Navigation and routing

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web application framework
- **PostgreSQL** - Robust relational database
- **JWT** - Authentication and authorization

### Payment Integration
- **PayFast** - South African payment gateway
- **Yoco** - Alternative payment option
